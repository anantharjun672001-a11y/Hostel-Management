# Hostel Management System (MERN)

A full stack Hostel Management System built using MERN stack.

## Live Demo
https://stayhive-hostel.vercel.app/

## Tech Stack
- React (Vite)
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- TailwindCSS

## Features

- Role based authentication
- Admin dashboard
- Resident management
- Room allocation
- Maintenance requests
- Rent payments
- Notifications
- Billing system

## Installation

Clone the repository

git clone (https://github.com/anantharjun672001-a11y/Hostel-Management)

Backend setup

cd backend
npm install

Frontend setup

cd client
npm install
